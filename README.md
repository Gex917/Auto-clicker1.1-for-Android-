# 🎮 自动点击器 (AutoClicker)

一款 Android 自动点击应用，适用于关卡类游戏的重复性操作。

## ✨ 功能特点

- 🎯 自定义点击位置坐标
- ⏱️ 设置点击间隔和等待时间
- 🔁 支持有限次数或无限循环
- 🪟 悬浮窗控制面板（开始/停止）
- 📱 简洁易用的界面
- 🔒 使用无障碍服务，无需 Root

## 📋 系统要求

- Android 7.0 (API 24) 及以上
- 需要开启无障碍服务权限
- 需要开启悬浮窗权限

## 🛠️ 编译方法

### 方法一：使用 Android Studio（推荐）

1. 下载并安装 [Android Studio](https://developer.android.com/studio)
2. 打开 Android Studio，选择 "Open an existing project"
3. 选择 `AutoClicker` 文件夹
4. 等待 Gradle 同步完成
5. 点击 "Build" → "Build Bundle(s) / APK(s)" → "Build APK(s)"
6. 编译完成后，APK 位于 `app/build/outputs/apk/debug/app-debug.apk`

### 方法二：使用命令行

```bash
# 进入项目目录
cd AutoClicker

# Windows
gradlew.bat assembleDebug

# Mac/Linux
./gradlew assembleDebug
```

## 📖 使用说明

### 第一步：安装应用

将编译好的 APK 传输到手机并安装。

### 第二步：开启权限

1. **无障碍服务**
   - 打开手机设置 → 无障碍 → 找到「自动点击器」→ 开启

2. **悬浮窗权限**
   - 打开手机设置 → 应用管理 → 自动点击器 → 悬浮窗 → 允许

### 第三步：设置点击位置

1. 打开应用
2. 点击「添加点击位置」
3. 输入屏幕坐标（X, Y）

**如何获取屏幕坐标：**
- 打开手机「设置」→「开发者选项」→ 开启「指针位置」
- 手指触摸屏幕，顶部会显示坐标

### 第四步：配置参数

- **重复次数**：执行几轮点击（0 = 无限循环）
- **点击间隔**：每次点击之间的等待时间（毫秒）
- **轮次间隔**：每轮执行完后的等待时间（毫秒）

### 第五步：启动

1. 点击「启动自动点击」按钮
2. 屏幕上会出现悬浮窗控制面板
3. 点击「开始」按钮开始自动点击
4. 点击「停止」按钮暂停
5. 点击「关闭」按钮退出

## ❓ 常见问题

**Q: 点击没反应？**
A: 请确保已开启无障碍服务权限。

**Q: 悬浮窗不显示？**
A: 请检查是否授予了悬浮窗权限。

**Q: 点击位置不准确？**
A: 不同手机分辨率不同，请用「指针位置」重新获取坐标。

**Q: 如何停止脚本？**
A: 点击悬浮窗的「停止」按钮，或关闭应用。

**Q: 需要 Root 吗？**
A: 不需要！使用无障碍服务即可。

## ⚠️ 注意事项

1. 请合理使用，避免影响游戏平衡
2. 部分游戏可能检测自动化操作
3. 建议在单机游戏中使用
4. 使用前请了解游戏的相关规定
